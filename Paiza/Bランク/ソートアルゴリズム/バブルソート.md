

## 学んだこと
バブルソートは、データ列の隣り合う要素を比較し交換することを繰り返すことによりデータ列をソートする手法です。バブルとは「泡」の意味で、ソートの過程でデータが移動する様子が、水中で泡が浮かんでいくように見えることからこの名前がついています。


## 模範解答
```python
def bubble_sort(a, n):
    for i in range(0, n - 1):
        for j in range(n - 1, i, -1):
            if a[j - 1] > a[j]:
                a[j - 1], a[j] = a[j], a[j - 1]

        print(*a)


n = int(input())
a = list(map(int, input().split()))

bubble_sort(a, n)
```
