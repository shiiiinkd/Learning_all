挿入ソートの実装

## 模範解答
```python
def insertion_sort(a, n):
    for i in range(1, n):
        x = a[i]
        j = i - 1

        while j >= 0 and a[j] > x:
            a[j + 1] = a[j]
            j -= 1

        a[j + 1] = x

        print(*a)


n = int(input())
a = list(map(int, input().split()))

insertion_sort(a, n)
```
