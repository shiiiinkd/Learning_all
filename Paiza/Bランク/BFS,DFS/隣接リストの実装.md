

## 学んだこと
隣接リストの概念と実装
- 各頂点に対して、「そこからつながっている頂点の一覧」を記録したもの
- つまり、  
    頂点 `u` → `[隣接する頂点のリスト]  
    の形で表現する
```
例
1 — 2 — 3
|    \
4     5

出力
1: [2, 4]
2: [1, 3, 5]
3: [2]
4: [1]
5: [2]
```
幅優先探索や深さ優先探索に応用でき、最短経路問題や連結成分判定などに使用する
メモリ使用量：`O(V + E)（辺の数に比例）
- 頂点の数Vに加え、各辺の両端を記録するためEに比例
探索時間：
- 隣接頂点の確認はO(V + E)  
- 隣接行列（O(V²)）に比べて大幅に効率的

## 隣接リストの実装（無向グラフを隣接リストで表す）
```python
# 入力
n, m = map(int, input().split())

# 普通の辞書を使って隣接リストを初期化
# -> from collections import defaultdict を用いると初期化の処理を省略できる

graph = {} # dictで隣接リストの関係を管理する

# 辺を読み込んで隣接リストを作る
for _ in range(m):
    a, b = map(int, input().split())
    
    # aの隣接リストが未作成なら空リストを作る
    if a not in graph:
        graph[a] = []
    if b not in graph:
        graph[b] = []
    
    # 無向グラフなので両方向に追加
    graph[a].append(b)
    graph[b].append(a)

# 1〜nの各頂点について出力
for i in range(1, n + 1):
    if i in graph:
        graph[i].sort() # 隣接する頂点の番号をソートして見やすくする
        print(f"{i}: {graph[i]}")
    else:
        # 隣接する頂点がない場合も空リストとして出力（孤立した頂点）
        print(f"{i}: []")

```

## コードの意味
- 「頂点 → つながっている頂点の一覧」という対応関係を辞書（`dict`）で保持している
- pythonの辞書は存在しないキーを参照するとエラーになる→キーがなければ空リストで初期化
```python
graph[a].append(b)
graph[b].append(a)
```
- 無向グラフのため双方向追加。隣接リストでグラフのつながりを明示的に記録する
```python
graph[2] = [5, 3, 1]
graph[2].sort()  # [1, 3, 5] に並び替え
```
- ソートすると番号の昇順に並び替えられる（デフォルトは入力順）
