## 問題内容
九九の表を各段で改行して出力
## 自分の解答
```python
a = [i+1 for i in range(9)] #そもそもこんな準備は必要ない
for j in range(9):
    for i in range(9):
        a[i]*=(j+1)
    print(*a)
    for i in range(9):
        a[i]//=(j+1)
```

## 模範解答
```python
for i in range(1, 10):
    for j in range(1, 10):
        print(i * j, end="")
        if j != 9:
            print(" ", end="")
    print() #改行出力
```
